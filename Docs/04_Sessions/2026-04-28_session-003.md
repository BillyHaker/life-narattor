# 2026-04-28 Session 003 — AI Review home entry polish

## Goal
Make the AI Review home screen feel less like a traditional search page and more like a lightweight AI-native review entry: users can ask directly, start from real clues discovered in their records, or use a small set of example prompts.

## Files Changed
- `Life Narattor/Screens/SearchScreen.swift`
- `Docs/VERIFICATION_BACKLOG.md`
- `Docs/04_Sessions/2026-04-28_session-003.md`
- `Docs/05_Changes/Change-211-ai-review-home-entry-polish.md`

## Implementation Notes
- Reframed the input card around `想从最近的记录里看见什么？` so the page clearly communicates reflection/discovery rather than keyword search.
- Changed the primary action from `回看` to `开始回顾`.
- Moved real clue suggestions above generic example prompts when suggestions exist.
- Renamed the clue section to `从这些线索开始` and explained that clues come from recent records.
- Reduced the example prompt section to an auxiliary `试着这样问` block.
- Polished clue cards into quieter iOS-style list entries: softer background, smaller color usage, chevron affordance, two-line title support, and user-facing `片段` wording instead of backend-like `材料`.
- Preserved existing behavior: tapping a clue still selects its tag type, sets the range to recent 90 days, fills the prompt, and immediately starts AI Review.

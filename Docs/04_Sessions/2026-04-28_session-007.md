# 2026-04-28 Session 007 — Timeline date header alignment

## Goal
Make the Timeline page top structure consistent with the Record page by adding the same `今天 · yyyy/MM/dd` date header above the range filter, while keeping the page lightweight and avoiding extra explanatory copy.

## Files Changed
- `Life Narattor/Screens/TimelineScreen.swift`
- `Docs/VERIFICATION_BACKLOG.md`
- `Docs/04_Sessions/2026-04-28_session-007.md`
- `Docs/05_Changes/Change-215-timeline-date-header-alignment.md`

## Implementation Notes
- Added a minimal Timeline `headerView` showing `今天 · yyyy/MM/dd`.
- Placed the header immediately above the Timeline segmented range picker.
- Tightened the Timeline root vertical spacing from 16 to 14 so the date and filter read as one top control group.
- Added `formattedTodayDate(_:)` on `TimelineScreen` instead of reusing the day-card date formatter, preserving day cards like `4月24日 · 周五`.
- Kept picker size native. The decision is to rely on Apple's segmented-control height and tune comfort through 16pt horizontal margins plus 14pt group spacing, rather than forcing a custom larger control.
- No filtering, snapshot, summary, or data behavior was changed.

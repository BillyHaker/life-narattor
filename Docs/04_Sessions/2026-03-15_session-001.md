# Session Log

## Goal
将拆分链路从“强制 typed atoms”收口为“record unit first”，移除 AI 输出中的类型枚举约束，避免类型分类反向驱动碎拆。

## Plan
1. 去掉 atomize schema 中对 `type` 的强制要求。
2. 清理 atomize payload 中仍残留的旧策略字段。
3. 保留 UI 兼容层，通过本地推断粗类型维持现有展示与 icon。
4. 运行静态检查与 build，确认链路一致。

## Work Log
- 在 `AIService.swift` 中移除 backend atomize 请求里的旧字段：`keep_user_voice`、`prefer_atomic_meaning_units`。
- 将 atomize policy 统一为 `no_formalization + max_atoms + prefer_retainable_units`。
- 让 `AtomDraft` 在缺少 `type` 时可解码，并通过 `AtomType.inferred(from:)` 本地推断粗类型，避免 UI 立即断裂。
- 同步 backend `/v1/atomize` schema 为 content-first，不再强制模型先分类再拆分。

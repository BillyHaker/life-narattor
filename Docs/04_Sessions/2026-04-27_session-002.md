## Implementation Notes
- Release/TestFlight no longer falls back to a local OpenAI API key when backend config is absent.
- Release/TestFlight transcription no longer silently falls back to local speech after AI/backend failure; this avoids false success in beta.
- Transcription provider defaults to Doubao in backend config and docs.
- Dev Tab remains debug-only via existing `#if DEBUG`; docs now state this explicitly.
- Old ReviewHome weekly/monthly entry points were removed from that screen.
- Legacy weekly/monthly review screens now use rolling `7 天回顾` / `30 天回顾` semantics and copy if reused.
- Timeline copy now says storylines appear when already prepared instead of implying guaranteed realtime generation.
- Removed remaining mock provider test expectation and updated provider label test to use unavailable service.
- Added fixed AI Review evaluation samples with explicit pass/fail criteria.

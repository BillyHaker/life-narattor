## Implementation Notes
- Removed `RootTab.projects` and the standalone bottom `线索` tab from the main `TabView`.
- Kept `ProjectsListScreen` and `ProjectDetailScreen` in the codebase for future secondary use; they are no longer a primary navigation entry.
- Updated the AI Review header copy from a pure search prompt to a lighter `想回看什么？` entry.
- Added a `最近线索` inspiration section to the AI Review empty/home state.
- The section only surfaces user-visible tags that already have related atom material, sorted by material count and recency.
- Each clue appears as a compact, tappable card with tag type, material count, and a light icon/tint treatment.
- Tapping a clue automatically selects the matching tag filter, sets the range to `最近90天`, fills a natural question, and triggers AI Review.
- Updated App Store submission copy so the visible primary tabs no longer mention `项目`.

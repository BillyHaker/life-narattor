# Seed Pack North Star

## Purpose
This seed pack is designed to be reused across new app projects. It provides:
1) Universal industry norms (process, governance, quality gates, privacy, contracts).
2) Meta-skills to help the user think through a new product and then generate project-specific Skills and a SKILLS_INDEX.

## Principles
- Specs live in Skills. Traceability lives in Docs. Enforcement lives in Rules.
- Keep this seed pack project-agnostic. Generate project specifics into a new project Skills folder.
- Prefer simple language and choice menus for non-technical owners.

## Outputs expected from meta-skills
- PROJECT_BLUEPRINT.md (product + UI + AI + data decisions)
- Project Skills library + a project SKILLS_INDEX.md
- Standard Docs/ + Templates/ + Rules/ scaffolding guidance

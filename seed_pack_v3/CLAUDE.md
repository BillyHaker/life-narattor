# Claude / Agent Entry Point (Seed Pack)

- Follow `Rules/AI_RULES.md` and `Rules/DEV_LOG_RULES.md`.
- Use `Skills/SKILLS_INDEX.md` to locate meta-skills and run them in order.

Generate project-specific skills under `project_skills/<project_name>/` instead.

Use `Skills/kickoff-prompt-generator/SKILL.md` to generate the kickoff prompt for this project.

_# Micro Mentoring - Seadance 2.0 最终版中文提示词 (15秒完整版)_

**使用说明**：
- 每个 Shot 对应一个独立的15秒提示词，直接复制使用。
- 在提示词中 `@探索者` `@领航员` `@异界导师` `@社群` 的位置，请上传对应的参考图。
- **字幕高亮**：在“约束”部分已标注需要在后期制作中高亮的关键词，以致敬PIRLS价值观。

### 🎞️ Shot 1｜选择之前：混沌的世界 (0–15s)

```
主体: @探索者，一个年轻的东亚男性，戴着眼镜，身穿深色斗篷。
动作: 
- (0-5s) 悬浮在一个巨大的黑暗虚空中，周围漂浮着发光的数据碎片和符文。
- (5-10s) 伸出手，指尖轻轻触碰一个漂浮的光点，光点在黑暗中泛起涟漪。
- (10-15s) 眼神专注而有思想，眼睛里反射出流动的数据符号和光点。
镜头: 
- (0-5s) 广角镜头，缓慢向前推进。
- (5-10s) 切换到中景镜头，固定机位，焦点在手上。
- (10-15s) 切换到特写镜头，极其缓慢地放大眼睛。
风格: 电影感，蓝紫色星云光效，柔和的背光，类似《银翼杀手2049》的氛围。
旁白: "Every journey begins with a **Passion**... a question that drives us." (低沉、沉思的男声)
约束: 无额外人物，无文字，保持人物居中，总时长15秒。字幕高亮: **Passion** (金色)。
```

### 🎞️ Shot 4｜迷雾被命名：Translation Gap (45–60s)

```
主体: @探索者 和数字迷雾。
动作: 
- (0-5s) @探索者 走向翻滚的数字迷雾，迷雾中是复杂的符文和数据洪流。
- (5-10s) 试图用手拨开迷雾，但迷雾只是暂时散开又迅速合拢，他显得有些沮丧。
- (10-15s) 复杂、无法解读的符文在迷雾中快速闪烁、变化。

- (0-5s) 广角镜头，固定机位。
- (5-10s) 中景镜头，固定机位。
- (10-15s) 切换到特写镜头，快速剪辑，营造混乱感。
风格: 电影感，迷雾压迫感强，强调徒劳无功的感觉。
旁白: "Knowledge alone creates distance. **Inclusion** is how we build the bridge." (略带沮丧和反思的男声)
约束: 无额外人物，无文字，总时长15秒。字幕高亮: **Inclusion** (蓝绿色)。

### 🎞️ Shot 7｜关键转折：复杂 → 清晰 (90–105s)

```
主体: @探索者 和符文能量球。
动作: 
- (0-5s) @探索者 透过水晶透镜看向那团符文能量球。
- (5-10s) 透过透镜的视角：混乱复杂的符文被过滤、简化，最终压缩成一个纯净、明亮的光点。

- (10-15s) 切换到广角镜头，固定机位。
风格: 电影感，从混乱到清晰的视觉转变过程，效果神奇，突破时刻，豁然开朗。
旁白: "Clarity is not about saying more. It is about showing what truly matters." (顿悟、清晰有力的男声)

# Progress Gates & Checkpoints (Meta)

## Gates
- Gate 0: Repo bootstrap present (Rules/Skills/Docs/Templates)
- Gate 1: Skills coverage complete (matrix + index)
- Gate 2: Contracts ready (AI I/O + DB + UI states)
- Gate 3: Buildable skeleton (app shell + mocks)
- Gate 4: Handoff ready (logs + ADR + changelog)

## Required output
A coverage report listing missing skills/sections until complete.

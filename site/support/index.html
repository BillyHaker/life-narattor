Life Narattor 技术支持

如果你在测试版中遇到问题，可以按下面方式反馈。

1. 当前测试范围

  • 文本记录
  • 语音记录与转写
  • 助手对话
  • 对话整理为记录草稿
  • AI 回顾

2. 建议反馈内容

  • 问题发生的页面和步骤
  • 是否与语音、AI 回顾或助手有关
  • 大概发生时间
  • 如果方便，请附截图

3. 联系方式

测试期建议通过你收到邀请码时使用的联系渠道反馈。如果你准备公开这页，请把这里替换成你的实际支持邮箱或 GitHub Issues 地址。

推荐填写方式示例：

  • support@your-domain.com
  • https://github.com/<your-org>/<your-repo>/issues

4. 提示

语音转写、助手和 AI 回顾依赖联网服务。如果遇到超时、失败或网络波动，请一并说明当时的网络环境。

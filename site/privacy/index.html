Life Narattor 隐私政策

更新日期：2026-03-29

1. 默认存储方式

我们重视你的隐私。默认情况下，你记录的内容只保存在你的本地设备上，包括文字记录、语音、转写和整理结果。我们不会默认将这些原始记录内容上传到服务器做长期存储。

2. 什么时候会联网

当你主动使用语音转写、助手对话、AI 回顾等需要联网处理的功能时，应用会通过产品后台代理调用 AI 能力。该过程中只会发送当前功能所必需的数据，不会把上游 AI 服务提供方的密钥暴露给终端用户。

3. 服务端默认保存哪些信息

  • 测试用户身份标识
  • 设备关联信息
  • API 使用量与额度信息
  • 匿名化产品统计与错误诊断信息

服务端默认不将完整记录正文、完整转写结果和完整草稿作为长期内容主库保存。

4. 测试期说明

当前版本为测试版。部分 AI 功能依赖联网服务，处理速度和结果质量可能受到网络环境影响。我们会根据测试反馈持续优化功能与隐私边界说明。

5. 未来能力

如果未来版本增加云同步或用户自带 AI API 等能力，我们会在功能启用前单独说明对应的数据处理方式，并由你主动选择是否开启。

6. 联系我们

如果你对隐私处理方式有问题，请通过 技术支持页面 反馈。
